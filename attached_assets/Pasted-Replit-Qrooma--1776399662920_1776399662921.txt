Replitへ

追加修正です。  
今のQroomaは見た目が静かになってきた一方で、動きが少なすぎて、少し止まって見える 状態です。  
今回は、派手さではなく、上質さを感じる微細なモーション を入れてください。

## 目的
- UIに少しだけ生命感を出す
- ただし、AIっぽい派手な演出にはしない
- Appleっぽい静かなプロダクト感を壊さない
- “動いている” ではなく “丁寧” と感じるレベルにする

---

## 基本方針
### 入れたい動き
- fade
- slight translate
- opacity transition
- soft scale
- smooth expand / collapse

### 入れたくない動き
- bounce
- strong zoom
- flashy slide-in
- neon / glow
- exaggerated parallax

---

## 具体的に入れてほしいモーション

### 1. Hero / セクション表示
ランディングの各セクションが表示されるときに、
軽い fade-up を入れてください。

#### 方向
- opacity: 0 → 1
- translateY: 8px → 0
- duration: 400〜700ms くらい
- easing は自然なもの
- セクションごとに微妙なずれがあってもよいが、やりすぎない

#### 対象
- Hero
- 訴求カード
- 使い方
- モード紹介
- BYOK説明
- フッターCTA

---

### 2. カード hover / tap
カードが静的すぎるので、ごく小さい反応 を入れてください。

#### 方向
- hover時に 1〜2px 浮く
- shadow がほんの少しだけ増える
- border / background が少しだけ変わる
- mobile では tap 時に自然な反応がある程度でよい

#### 対象
- Landing の訴求カード
- Rooms 一覧カード
- Settings の各設定カード
- API説明カード
- 結論カード

---

### 3. CTA / ボタン
ボタンが今は少し静止しすぎています。  
上質な反応 を入れてください。

#### 方向
- hover: opacity / background / translateY を少しだけ
- active / press: scale 0.98 前後
- transition は短めで気持ちよく
- locale切替ボタンも、選択状態の変化が滑らかに見えるようにする

#### 対象
- Hero CTA
- Nav CTA
- 新しいルーム
- APIキー取得
- ログイン / はじめる
- 言語切替トグル

---

### 4. アコーディオン / ドロワー / モーダル
開閉がパキッとしすぎると安っぽく見えるので、なめらかに してください。

#### 対象
- 結論カードの開閉
- サイドバー（モバイルドロワー）
- 初回案内やAPI説明モーダルがある場合はその開閉

#### 方向
- opacity + height / translate の自然な transition
- 速すぎず遅すぎず
- 見ていて邪魔にならない程度

---

### 5. locale切替 / 状態変化
日本語 / 英語切替や、状態表示の切替時に、突然切り替わる感じ を少し和らげたいです。

#### 方向
- text は瞬間でよいが、周辺要素の opacity / background / pill state の変化は少しだけ滑らかに
- CTA ボタン文言崩れが出ないように、locale切替時の再描画は安定させる
- “アニメーションのために壊れる” は絶対に避ける

---

## motion を入れる優先箇所
### P0
1. Hero / landing sections の fade-up
2. ボタンの hover / press
3. 結論カードの開閉
4. モバイルサイドバーの開閉
5. Rooms 一覧カードの軽い hover / tap

### P1
1. Settings カードの hover
2. locale切替の滑らかさ
3. 状態 pill / badge の transition

---

## 重要な注意
- motion は主役にしない
- 動きで目立たせるのではなく、完成度を上げるために使う
- “静かだけど気持ちいい” が正解
- 全体で統一感を持たせる
- duration / easing / hover量をバラバラにしない

---

## 完了条件
1. ランディングが少しだけ気持ちよく動く
2. カードやボタンに上質な反応がある
3. アコーディオンやドロワーがなめらか
4. locale切替時に壊れず、違和感が少ない
5. 派手すぎる演出は入っていない
