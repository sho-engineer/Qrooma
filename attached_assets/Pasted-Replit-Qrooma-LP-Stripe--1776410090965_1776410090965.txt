プラン方針（fix）

Qrooma は 3プラン構成 にします。

1. Free
	•	月額 $0
	•	APIキー不要
	•	Qrooma側の固定構成で使える体験版
	•	無料枠には利用制限あり
	•	まず価値を体験してもらうためのプラン

Free の内容
	•	2エージェントまで
	•	モデル選択不可
	•	1日 3回までを想定
	•	1回あたり短めの議論
	•	自動結論あり
	•	体験版として使える
	•	本格利用には Connect または Pro へ進む

LP / UIで必ず伝えること
	•	無料枠には利用制限があります
	•	APIキーなしで試せる
	•	本格利用やモデル選択には上位プランまたはAPIキー設定が必要

⸻

2. Connect
	•	月額 $9
	•	APIキー必要
	•	いわゆる BYOK プラン
	•	本命の個人向けプラン

Connect の内容
	•	自分のAPIキーで利用
	•	2 / 3 エージェント選択可
	•	モデル選択可
	•	同一 provider + model の重複不可
	•	ディベート / フリートーク両方
	•	自動結論あり
	•	手動で ここまでをまとめる / 結論を出す も将来対応前提
	•	実務利用向け

LP / UIで伝えること
	•	Bring your own API keys
	•	自分の契約で本格利用できる
	•	モデルと構成を選べる
	•	原価を抑えつつ柔軟に使える

⸻

3. Pro
	•	月額 $20
	•	APIキー不要
	•	Qrooma 側が利用を含むプラン
	•	月額料金に一定量の利用を含む
	•	上限を超えたら従量課金

Pro の内容
	•	APIキー不要
	•	月30回の議論を含む前提で見せる
	•	超過分は追加課金
	•	2 / 3 エージェント利用可
	•	モデルは固定または一部選択可でもよい
	•	非ITの人にも導入しやすい
	•	“すぐ使える有料プラン” の位置づけ

LP / UIで伝えること
	•	Includes usage
	•	30 discussions / month included
	•	Additional usage is billed separately
	•	APIキー不要でそのまま使える
	•	本格的に使いたい人向け

	•	Connect: APIキーで本格利用

2. APIキー説明を最新化する

今後は「全員が必ずAPIキー必須」ではなくなります。
そのため、説明文を以下に更新してください。

日本語
	•	Free では APIキーなしで試せます
	•	Connect では APIキー設定で本格利用できます
	•	Pro では APIキー不要で利用できます
	•	APIキーを設定すると、モデルや構成を柔軟に選べます

	•	Connect lets you use your own API keys for full flexibility
	•	Pro includes usage, so no API keys are required
	•	Adding API keys unlocks more control over models and setup

Settings には保存ボタンを入れてください。

方針:
