Replitへ

Qrooma の見た目を、Appleっぽいモダンで静かなUI に寄せてください。
ここで言う Apple は 機能や情報設計ではなく、見た目の方向性 の話です。
機能追加は不要 で、今回は ビジュアル改善とデザイン統一 が目的です。

目的
	•	今の「AI感が強い」印象を減らす
	•	派手さより、静かで洗練されたSaaS / Apple-like な雰囲気にする
	•	Qrooma の価値が、見た目のノイズで埋もれないようにする
	•	ランディングページとアプリ本体のデザイン言語を揃える

前提
	•	ルーティングや基本構成は変えない
	•	UXフローも大きく変えない
	•	追加機能は不要
	•	見た目のみを改善する

全体デザイン方針

以下の方向に寄せてください。

トーン
	•	クリーン
	•	ミニマル
	•	落ち着いている
	•	信頼感がある
	•	余白が広い
	•	情報に呼吸感がある
	•	“AIツールっぽさ” より “完成度の高いプロダクト感”

やってほしいこと
	•	色数を減らす
	•	強いグラデーションやネオン感を減らす
	•	過剰なシャドウを弱める
	•	バッジや強調色の使いすぎを抑える
	•	余白とタイポグラフィで見せる
	•	同一トークン（radius / spacing / border / shadow）で全画面を統一する

避けたいこと
	•	サイバー感
	•	派手なAI感
	•	強い発光
	•	過度なグラスモーフィズム
	•	不必要なアニメーション
	•	色で情報を出しすぎること

⸻

ビジュアル具体指示

1. カラーパレット
	•	ベースは 白 / ごく薄いグレー / 黒に近いテキスト
	•	アクセントは 控えめなブルー系 を1つ主軸に
	•	ChatGPT / Claude / Gemini の色は残してよいが、主役にしない
	•	モデル色は
	•	小さなドット
	•	薄いライン
	•	控えめなアバターアクセント
くらいに抑える

2. タイポグラフィ
	•	できるだけ Apple的な読みやすいタイポ に寄せる
	•	見出しは太め、本文は細すぎず読みやすく
	•	情報の階層を、色ではなく 文字サイズ・ウェイト・余白 で出す
	•	英文 / 日本語ともに自然に見えるバランスにする

3. カード
	•	角丸は上品に（やや大きめでもOK）
	•	ボーダーはごく薄く
	•	シャドウは薄く、浮きすぎない
	•	カードの中身は詰め込みすぎず、余白を広めに

4. ステータス表現
	•	完了 / 実行中 / エラー / 待機中 は、今より静かな表現にする
	•	強い色ベタではなく、
	•	ソフトなピル
	•	薄い背景色
	•	控えめな文字色
に寄せる
	•	とくにエラーも「壊れている感」が出すぎないようにする

5. ボタン
	•	主CTAは明確に
	•	Secondary / tertiary は軽く
	•	ボタンサイズ、角丸、余白を全画面で統一する
	•	派手なグラデーション不要

⸻

ページごとの指示

ランディングページ
	•	Appleっぽく、文字と余白で見せる
	•	Hero は大きく、静かに、力強く
	•	セクション間の余白を広めに
	•	訴求カードは今よりシンプルに
	•	“売り込み感” より “完成されたプロダクト紹介” に寄せる
	•	CTA は明確にするが、うるさくしない

ログイン / サインアップ
	•	クリーンで安心感のある認証画面にする
	•	デモモードの表示も、警告感より説明感
	•	余白をしっかり取り、フォームの完成度を上げる

サイドバー
	•	より静かで洗練された見た目に
	•	アイコンのみ表示の折りたたみ状態も美しく
	•	モバイルドロワーもノイズなく整理
	•	アクティブ状態は控えめだが分かりやすく

ルーム一覧
	•	カードの完成度を上げる
	•	情報は整理して、詰まって見えないように
	•	エラーバッジも落ち着いた表現にする

ルーム詳細
	•	ここが一番重要です
	•	“AIチャットっぽさ” を少し減らして、
思考の場 / ディスカッションの場 に見えるようにする
	•	バブル、区切り線、結論カード、ThinkingIndicator を統一された静かな見た目にする
	•	モデル色は補助的に扱う
	•	全体として「議論の内容」が主役に見えるようにする

結論カード
	•	より重要で信頼できる情報に見えるように
	•	派手さより、読みやすさと整理感を重視
	•	アコーディオンや要点リストも上品に

設定画面
	•	フォームUIをより洗練させる
	•	APIキー未設定バッジは残してよいが、警告が強すぎないようにする
	•	1画面の密度をやや下げ、読みやすくする

⸻

役割上の補足

今回は Apple 風 ＝ 見た目の話だけ です。
Qrooma の中身や価値は維持したまま、表面の空気感を洗練させる ことが目的です。
